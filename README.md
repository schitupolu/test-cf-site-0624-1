## Quick start

Quick start options:

- Downdload the ZIP project.
- Extract the files from the ZIP project.

## Terminal Commands

1. Download and Install NodeJs LTS version from [NodeJs Official Page](https://nodejs.org/en/download/).
2. Navigate to the root ./ directory of the product and run `npm install` or `yarn install`.
3. Execute the command `npm run dev` to run the template in your localhost.
